# Skill Export Archives

Send one of these archives to your friend:

- `hackathon-skills-export.zip` for Windows or general use
- `hackathon-skills-export.tar.gz` for macOS/Linux

After extracting, your friend can run:

```bash
chmod +x install.sh
./install.sh
```

On Windows PowerShell:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\install.ps1
```

The installer copies the exported skills to `~/.codex/skills/hackathon-skills` by default.
